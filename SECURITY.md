# Security Policy

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability. Use the repository's
**Security** tab and select **Report a vulnerability** to submit a private GitHub Security
Advisory. Include reproduction steps, affected versions, and the impact you observed.

This project is currently a private beta. There is no guaranteed response SLA, but reports will
be acknowledged and assessed as quickly as possible. Please allow time for a fix before publicly
disclosing an issue.

## Supported versions

Only the latest commit on the default branch is supported during the private beta.
